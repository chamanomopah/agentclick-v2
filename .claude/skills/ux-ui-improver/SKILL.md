## Analysis Framework

### 1. Usability Assessment
- Can users accomplish their goals efficiently?
- Is the navigation intuitive?
- Are interactive elements clearly identifiable?

### 2. Visual Design
- Is the visual hierarchy clear?
- Do colors, typography, and spacing work well together?
- Is the design consistent throughout?

### 3. Accessibility
- Are there proper contrast ratios?
- Is the interface keyboard-navigable?
- Are screen readers supported?

### 4. Mobile Responsiveness
- Does the design work on different screen sizes?
- Are touch targets appropriately sized?
- Is content prioritized on mobile?

### 5. Performance
- Do animations run smoothly?
- Is load time acceptable?
- Are images optimized?

## Output Format

Provide:
1. **Strengths**: What's working well
2. **Issues**: Specific problems found
3. **Recommendations**: Actionable improvements
4. **Code Examples**: Specific implementation suggestions if applicable
5. **Priority**: Rank improvements by impact vs effort
